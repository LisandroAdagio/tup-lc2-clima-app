Seccion de Ayuda de ClimaApp

Esta APP permite consultar el clima de una ciudad en particularutilizando la

API de OpenWeatherMap.

Para poder consultar el clima, primero deberá AgregarUnaCiudad y luego

seleccionar la ciudad desde la página Clima.
